Automated Weekly Reports — GA4 → PowerPoint

Built a Python pipeline that connects to Google Analytics 4 API, extracts key metrics weekly, and automatically generates a formatted PowerPoint report — eliminating hours of manual reporting work.

How it works: The script authenticates via Google API, pulls sessions, users, bounce rate, conversions and traffic sources, then formats everything into branded PowerPoint slides with charts and KPI summaries — sent automatically every Monday.

# Simplified workflow
from google.analytics.data import BetaAnalyticsDataClient
from pptx import Presentation

client = BetaAnalyticsDataClient()
response = client.run_report(request={...})
prs = Presentation()
# → Build slides with data
prs.save("weekly_report.pptx")
